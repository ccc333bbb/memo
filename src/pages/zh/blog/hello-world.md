# 你好世界！👋

欢迎来到我的个人备忘录网站！这里是我分享想法、项目和精选书签的地方。

## 期待什么

这个网站使用 [Astro](https://astro.build/) 构建，具有以下特色：

- **多语言支持**：提供英文、繁体中文和简体中文内容
- **博客文章**：详细的文章和想法
- **项目展示**：即将推出！
- **书签收藏**：即将推出！
- **想法专区**：即将推出！

## 关于这个网站

这个网站是我的个人知识中心，我在这里收集和整理对我重要的信息。它采用优美的水墨风格设计，并使用现代网页技术构建。

网站整合了 GitHub Issues 来简化内容管理，让我能够轻松地撰写、发布和维护多语言内容。
