# 歡迎來到 Astro 部落格

這是您的第一篇範例文章，恭喜您開始使用 Astro 撰寫部落格！

## 功能特色

- 🌍 多語言支持
- 📝 GitHub Issues 聯動
- 🚀 自動化工作流程
- 📱 響應式設計

## 開始使用

1. 使用 `npm run blog:create` 創建新文章
2. 編輯文章內容
3. 使用 `npm run blog:publish <issue-number>` 發布文章
4. 提交代碼時包含 `Closes #<issue-number>` 自動關閉 issue

祝您寫作愉快！
